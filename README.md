# RAZBLOK-ROBLOX
В России с 3 декабря заблокировали роблокс и  моментально создали для него обход.
 .Инструкция:

1.Запустите service.bat, проверьте, отключите ли фильтр Switch Game Filter и его
2.Запустите общий (FAKE TLS AUTO или AUTO ALT1-2-3) от имени администратора.
3.попробуйте открыть Roblox и присоединиться к любому плейсу
Инструкции:

1.запустите service.bat проверьте, отключен ли Switch Game Filter, включите его
2.запуск общего (поддельный TLS AUTO или AUTO ALT1-2-3) от администратора
3.попробуй открыть роблокс и присоединиться к любому месту
